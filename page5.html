Sophmore Work

In the seen above images you will see something that is called Block Coding. Block Coding is something that I had the pleasure of doing in my Sophmore Year of Highschool. The image on the left is an app that I made to recreate a GPS type app, in this app you can pin a location you want to travel to and also add pins for points of interest. This app allowed me to learn about embedding a map into a app and allow a user to explore the world based off of named locations and even coordinates for a location.

On the right side youll see another image from another project that I had done in my Sophmore year. This app was a game based on the armed forces. The purpose of this game was to answer a question that was given to a user for points, if a player gets one wrong their score gets reset to the start and so on. I was able to learn how to have user based interactions in a game and how to get those user interactions to work within the app/game also.
